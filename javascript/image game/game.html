The aim of the game is to match the hide picture pairs.When clicking on a box, it shows the hidden picture. When the next card is matching you have found a pair. That pair stays uncovered. If two cards are not matching, it is being covered up agian. The game is fininshed if you uncover all the pairs. Try to do it with as little tries as possible.
